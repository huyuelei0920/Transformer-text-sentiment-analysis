# Transformer 中文情感分析系统使用说明

## 项目简介

本项目是一个**基于纯 Transformer 架构**的中文情感分析系统，从零实现了 Transformer Encoder 用于文本分类任务。系统支持三分类情感识别：正面、负面、中性。

训练时自动对不均衡数据进行下采样平衡，无需预处理数据。

### 模型架构

- 多头自注意力机制 (Multi-Head Self-Attention)
- 位置编码 (Positional Encoding)
- 层归一化 (Layer Normalization)
- 前馈神经网络 (Feed-Forward Network)
- 残差连接 (Residual Connection)

### 模型参数

| 参数 | 值 |
|------|-----|
| d_model | 128 |
| num_heads | 4 |
| num_layers | 2 |
| d_ff | 256 |
| dropout | 0.3 |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 训练模型

```bash
python train_transformer_web.py
```

训练过程中自动执行：
- 从 `train.csv` 和 `validation.csv` 加载数据
- 对多数类（正面）进行下采样，平衡三类数据
- 显示训练/验证损失、准确率、F1 分数
- 自动保存最佳模型

### 3. 运行 Web 应用

```bash
streamlit run app_transformer_web.py
```

### 4. 命令行推理

```bash
# 演示模式
python inference.py

# 分析单个文本
python inference.py --text "今天天气真好"

# 交互模式
python inference.py --interactive
```

## 数据格式

训练数据为 CSV 格式，包含 `text` 和 `label` 两列：

| text | label |
|------|-------|
| 今天天气真好 | 1 |
| 这个产品太差了 | 0 |
| 还可以吧 | 2 |

标签说明：
- `0`: 负面 (Negative)
- `1`: 正面 (Positive)
- `2`: 中性 (Neutral)

## 模型架构详解

### 1. 位置编码 (PositionalEncoding)

使用正弦和余弦函数生成固定位置编码：

```
PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
```

### 2. 多头自注意力 (MultiHeadAttention)

- 将输入通过 Q、K、V 线性变换
- 分成多个头并行计算注意力
- 缩放点积注意力：Attention(Q,K,V) = softmax(QK^T / sqrt(d_k)) V

### 3. 前馈网络 (FeedForward)

```
FFN(x) = GELU(xW_1 + b_1)W_2 + b_2
```

### 4. 编码器层 (TransformerEncoderLayer)

```
x = LayerNorm(x + MultiHeadAttention(x))
x = LayerNorm(x + FeedForward(x))
```

## 性能指标

训练完成后，模型会输出以下指标：

- **准确率 (Accuracy)**: 正确预测的比例
- **F1 分数**: 精确率和召回率的调和平均
- **混淆矩阵**: 各类别的预测分布
- **分类报告**: 每个类别的精确率、召回率、F1

## 使用示例

### Python API

```python
from inference import create_predictor

# 创建预测器
predictor = create_predictor(
    model_dir='transformer_checkpoints',
    model_file='best_model.pt'
)

# 预测单个文本
result = predictor.predict("今天天气真好，心情很愉快")
print(f"情感: {result['label_name']}")
print(f"置信度: {result['confidence_percent']}")

# 批量预测
texts = ["这个产品很好", "太差了", "一般般"]
results = predictor.predict_batch(texts)
```

## 注意事项

1. **首次运行**：需要先训练模型或确保 `transformer_checkpoints/` 中已有模型文件
2. **GPU 支持**：自动检测并使用 GPU（如果可用）
3. **内存需求**：建议至少 4GB RAM
4. **数据量**：建议至少 1000 条标注数据以获得较好效果
5. **数据平衡**：训练脚本会自动下采样多数类，无需手动预处理

## 常见问题

### Q: 模型未找到错误
A: 请先运行 `python train_transformer_web.py` 训练模型

### Q: CUDA 内存不足
A: 减小 `batch_size` 或 `max_length`

### Q: 训练速度慢
A: 确保 GPU 可用，或减小模型参数
