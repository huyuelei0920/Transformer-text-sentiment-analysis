# Transformer 中文情感分析系统使用说明

## 📖 项目简介

本项目是一个**基于纯 Transformer 架构**的中文情感分析系统，从零实现了 Transformer Encoder用于文本分类任务。

### 🏗️ 模型架构

- **多头自注意力机制 (Multi-Head Self-Attention)**
- **位置编码 (Positional Encoding)**
- **层归一化 (Layer Normalization)**
- **前馈神经网络 (Feed-Forward Network)**
- **残差连接 (Residual Connection)**

### 📊 模型参数

| 参数 | 值 |
|------|-----|
| d_model | 256 |
| num_heads | 8 |
| num_layers | 4 |
| d_ff | 512 |
| dropout | 0.1 |
| 总参数量 | ~470万 |

## 📊 数据格式

训练数据应为 CSV 格式，包含以下列：

| sentence | label |
|----------|-------|
| 今天天气真好 | 1 |
| 这个产品太差了 | 0 |
| 还可以吧 | 2 |

标签说明：
- `0`: 负面 (Negative)
- `1`: 正面 (Positive)
- `2`: 中性 (Neutral)

## ⚙️ 配置参数

在 `train_transformer.py` 中可以修改以下配置：

```python
CONFIG = {
    # 模型配置
    'd_model': 256,        # 模型维度
    'num_heads': 8,        # 注意力头数
    'num_layers': 4,       # 编码器层数
    'd_ff': 512,           # 前馈网络维度
    'dropout': 0.1,        # Dropout 比率
    'max_length': 128,     # 最大序列长度
    
    # 训练配置
    'batch_size': 32,      # 批次大小
    'num_epochs': 20,      # 训练轮数
    'learning_rate': 1e-4, # 学习率
    
    # 分词器配置
    'max_vocab_size': 30000,  # 最大词汇表大小
    'min_freq': 2,            # 最小词频
    'use_word': False,        # 是否使用词级分词
}
```

## 🔧 模型架构详解

### 1.位置编码 (PositionalEncoding)

使用正弦和余弦函数生成固定位置编码：

```
PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
```

### 2. 多头自注意力 (MultiHeadAttention)

- 将输入通过 Q、K、V 线性变换
- 分成多个头并行计算注意力
- 缩放点积注意力：Attention(Q,K,V) = softmax(QK^T/√d_k)V

### 3. 前馈网络 (FeedForward)

```
FFN(x) = GELU(xW_1 + b_1)W_2 + b_2
```

### 4. 编码器层 (TransformerEncoderLayer)

```
x = LayerNorm(x + MultiHeadAttention(x))
x = LayerNorm(x + FeedForward(x))
```

## 📈 性能指标

训练完成后，模型会输出以下指标：

- **准确率 (Accuracy)**: 正确预测的比例
- **F1 分数**: 精确率和召回率的调和平均
- **混淆矩阵**: 各类别的预测分布
- **分类报告**: 每个类别的精确率、召回率、F1

## 🎯 使用示例

### Python API

```python
from inference import create_predictor

# 创建预测器
predictor = create_predictor(
    model_dir='transformer_checkpoints',
    model_file='best_model.pt'
)

# 预测单个文本
result = predictor.predict("今天天气真好，心情很愉快")
print(f"情感: {result['label_name']}")
print(f"置信度: {result['confidence_percent']}")

# 批量预测
texts = ["这个产品很好", "太差了", "一般般"]
results = predictor.predict_batch(texts)
```

## 🔍 与 BERT 的对比

| 特性 | 纯 Transformer | BERT |
|------|---------------|------|
| 架构 | 仅 Encoder | Encoder (12层) |
| 预训练 | 无 | 有 |
| 参数量 | ~470万 | ~1.1亿 |
| 训练速度 | 快 | 慢 |
| 推理速度 | 快 | 慢 |
| 准确率 | 依赖数据量 | 通常更高 |

## 📝 注意事项

1. **首次运行**：需要先运行 `train_transformer.py` 训练模型
2. **GPU 支持**：自动检测并使用 GPU（如果可用）
3. **内存需求**：建议至少 4GB RAM
4. **数据量**：建议至少1000条标注数据以获得较好效果

## 🐛 常见问题

### Q: 模型未找到错误
A: 请先运行 `python train_transformer.py` 训练模型

### Q: CUDA 内存不足
A: 减小 `batch_size` 或 `max_length`

### Q: 训练速度慢
A: 确保 GPU 可用，或减小模型参数
